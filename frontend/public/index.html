Welcome to ICU v1.0.0

Inventory Control Utility - Learning Project

Quick Navigation:

  • Items Management - Create, read, and delete inventory items
  • Purchases Management - Manage purchase orders and items
  • Reviews Management - Track inventory reviews and audits

About This Application:

ICU v1.0.0 is a learning project demonstrating microservices architecture with:

  • Frontend: Node.js + Express
  • Items Service: Python + Flask
  • Reviews Service: Java + Spring Boot
  • Database: MySQL
  • Mocking: WireMock
